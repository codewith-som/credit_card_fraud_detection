# 💳 Credit Card Fraud Detection API

A Machine Learning based Fraud Detection API built using **Python**, **Flask**, and **Scikit-learn**.

This project predicts whether a credit card transaction is **fraudulent** or **safe** using a trained Random Forest Classifier.

---

## 🚀 Features

- Fraud Detection using Machine Learning
- Flask REST API
- Random Forest Classifier
- Feature Engineering
- JSON Request & Response

---

## 🛠️ Tech Stack

- Python
- Flask
- Pandas
- NumPy
- Scikit-learn
